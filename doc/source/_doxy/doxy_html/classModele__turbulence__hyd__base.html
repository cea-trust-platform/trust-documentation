Detailed Description

qui representent un modele de turbulence pour les equations de Navier-Stokes. Il existe deja deux classes derivees qui representent le modele de turbulence (k,e) et le modele de turbulence sous maille. Ces deux modeles ont en commun le calcul d'une viscosite turbulente.

B.Mat. : la classe herite de support_champ_masse_volumique. en Front-Tracking (et plus tard en qc), elle fournit le mecanisme pour recuperer rho.
