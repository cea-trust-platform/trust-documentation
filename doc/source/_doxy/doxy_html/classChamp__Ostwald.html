Detailed Description

   de l'indice de structure.

Classe mere qui gere le champ utilise pour le fluide d'Ostwald dans les deux discretisations. Possede les fonctions generiques aux deux discretisations. Fait reference a un fluide d'Ostwald pour pouvoir utiliser les deux parametres du fluide d'Ostwald : K et N.
