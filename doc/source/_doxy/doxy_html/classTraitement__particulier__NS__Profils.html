Detailed Description

This class enables a particular treatment for computations which have a periodic BC (ie homogeneous direction) and on which we want profiles in a specific direction and position :

  • mean velocity profiles
  • fluctuations
See also
Navier_Stokes_Turbulent, Traitement_particulier_base, Traitement_particulier_VDF
