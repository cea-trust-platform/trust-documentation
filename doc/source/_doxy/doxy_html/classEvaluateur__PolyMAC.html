Detailed Description

Classe de base des evaluateurs PolyMAC. Cette classe n'appartient pas a la hierarchie des Objet_U. Cette classe porte une reference a un objet de type Domaine_PolyMAC et une reference a un objet de type Domaine_Cl_PolyMAC. Elle porte des tableaux locaux qui sont en fait des references aux tableaux de l'objet de type Domaine_PolyMAC (ces tableaux locaux n'existent pas en memoire).
