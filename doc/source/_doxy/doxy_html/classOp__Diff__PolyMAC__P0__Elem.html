Assembles the diffusion contribution to the linear system.

This is the core method that computes and adds the diffusion terms to both the matrix and the right-hand side vector. The method:

  • Computes diffusive fluxes across all faces using the phi_f coefficients
  • Handles various boundary conditions (Dirichlet, Neumann, heat exchange)
  • Assembles matrix coefficients for implicit treatment
  • Updates the right-hand side with explicit contributions
  • Stores boundary fluxes for post-processing
Parameters
matricesMap of matrices to be filled
secmemRight-hand side vector to update
semi_implMap of semi-implicit fields
