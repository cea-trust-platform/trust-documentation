◆ syncfile()

Sortie & EcrFicPartage::syncfile ( )
overridevirtual

Provoque l'ecriture sur disque des donnees accumulees sur les differents processeurs depuis le dernier appel a syncfile().

Les donnees sont ecrites dans l'ordre croissant des processeurs. Cette fonction doit etre appelee le meme nombre de fois sur tous les processeurs !

Exemple: processeur 0: processeur 1: file << "pe0 : 1" << finl; file << "pe1 : 1" << finl; file << "pe0 : 2" << finl; file << "pe1 : 2" << finl; file.syncfile(); file.syncfile(); file << "pe0 : 3" << finl; file << "pe1 : 3" << finl; file << "pe0 : 4" << finl; file.syncfile(); file.syncfile(); file << "pe0 : end" << finl; // le processeur 1 n'ecrit pas de donnees file.syncfile(); file.syncfile();

Contenu du fichier : pe0 : 1 pe0 : 2 pe1 : 1 pe1 : 2 pe0 : 3 pe0 : 4 pe1 : 3 pe0 : end
