Detailed Description

Methodes implementees :

  • eddy_viscosity(nu_t) -> remplit le DoubleTab "nu_t" tel que <u'_i u'_j> = - nu_t (dui / dx_j + duj / dx_i + (partie trace))
  • reynolds_stress(R_ij) -> remplit R_ij = < u'_i u'_j> (pour utilisation par un modele GGDH dans une autre equation) format : R_ij(e, n, i, j) avec e l'indice d'element, n l'indice de phase, (i, j) les composantes
  • k_over_eps(k_sur_eps) -> renvoie la quantite "k / epsilon" (utilisee par exemple dans le GGDH) Dans les deux cas, les tableaux doivent etre dimensionnes par la fonction appelante.
