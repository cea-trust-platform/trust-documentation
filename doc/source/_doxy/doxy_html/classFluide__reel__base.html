Detailed Description

ses proprietes:

  • viscosite cinematique, (mu)
  • viscosite dynamique, (nu)
  • masse volumique, (rho)
  • diffusivite, (alpha)
  • conductivite, (lambda)
  • capacite calorifique, (Cp)
  • dilatabilite thermique du constituant (beta_co)
