Detailed Description

Possede les proprietes d'un fluide incompressible plus: N : indice de structure K : consistance du fluide La viscosite dynamique dans ce cas depend de K, N et Grad(U) Nouveau fluide. c'est un fluide incompressible non newtonien. Il derive donc de fluide_Incompressible et possede quelques proprietes de plus (K, N) mu depend de K, N, et de grad(U)
