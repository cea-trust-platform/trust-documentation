Detailed Description

en definissant des methodes ajouter,calculer et mettre_a_jour qui ne font aucun calcul. Cette classe est utilisee lors de la definition d'operateurs negligeable par heritage multiple.

See also
Op_Diff_negligeable Op_Diff_K_Eps_negligeable Op_Conv_negligeable, Classe interface, hors hierarchie operateurs TrioU., Doit faire partie d'un heritage multiple pour etre, utile a la definition d'un operateur negligeable instanciable.
