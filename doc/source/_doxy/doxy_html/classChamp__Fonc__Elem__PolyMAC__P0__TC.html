Detailed Description

This field is a Champ_Fonc_Elem_PolyMAC_P0 with 1 value per element and per phase :

Champ_Fonc_Elem_PolyMAC_P0_TC::valeurs()(e, n) returns the value of phase n in element e The shear rate is calculated using shear rate = sqrt(2*Sij*Sij) Sij = 1/2(grad(u)+t grad(u))
